Project Summary & Technologies Used

Project Overview:

The Risk-Aware Portfolio Optimisation System (RAPOS) is a comprehensive web application designed to help investors make data-driven investment decisions through automated portfolio management, risk assessment, and real-time analytics. The project demonstrates understanding of frontend web development concepts including HTML5 semantic markup, CSS3 styling and animations, JavaScript DOM manipulation and validation, responsive design principles, and data visualization using Chart.js library.

Technologies & Tools:

  • HTML5: Semantic elements, forms, responsive structure
  • CSS3: Flexbox, Grid, Gradients, Animations, Transitions, Media Queries
  • JavaScript (ES6): Form validation, DOM manipulation, localStorage, URL parameters, event handling
  • Chart.js (v4): Doughnut charts, line charts, bubble charts for data visualization
  • Design Patterns: Card-based UI, responsive grids, sticky navigation
  • Version Control: File-based project structure

Key Features Implemented:

  • User authentication system with client-side validation
