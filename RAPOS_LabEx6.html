1. Test Strategy

1.1 Overview

RAPOS (Risk-Aware Portfolio Optimisation System) is a full-stack AI financial terminal. The test strategy covers all layers: unit tests on the LSTM model and FastAPI backend, integration tests across frontend–backend–database, and end-to-end UI tests using Selenium IDE. The goal is to ensure correctness, performance, and security before each Vercel deployment.

6. Deployment Scheme & Diagram

6.1 Deployment Architecture Description

RAPOS uses a 3-tier cloud deployment architecture. The Presentation Tier is hosted on Vercel's global CDN edge network. The Application Tier runs on Render.com as a Python FastAPI server with the LSTM model loaded in memory. The Data Tier uses Supabase (managed PostgreSQL) with Row-Level Security and Realtime WebSocket subscriptions.
